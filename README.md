# Expense Tracker in Python

This is a simple Python-based Expense Tracker application that helps users record and manage their daily expenses. 
Users can enter the expense name, amount, and date. All data is stored in a text file so that expenses can be viewed later.
This beginner-friendly project is useful for understanding Python file handling, loops, and user input.

## Features
- Add expenses with name, amount, and date  
- View all recorded expenses  
- Automatically stores data in expenses.txt  
- User-friendly and simple to use  

## Technologies Used
- Python
- Text file handling (.txt)

## How to Run the Project
1. Make sure Python is installed on your system.
2. Place `expenses.txt` in the same folder as the python script `main.py`.
3. Open a terminal in the project folder and run:
python src/main.py
4. Use the menu to add/view/save expenses.

## Project Files
- src/main.py — Main program
- src/expenses.txt — Stores expense records
- screenshots/ — project screenshots
- README.md — This file
